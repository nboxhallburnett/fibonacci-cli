Fibonacci-cli [![npm version](https://badge.fury.io/js/fibonacci-cli.svg)](https://www.npmjs.com/package/fibonacci-cli)
======

Prints out the fibonacci number at the index provided

### Usage

```
> fibonacci [index]
```

Example:
```
> fibonacci 10
89
```

### Install

To install, run:
```
npm install -g fibonacci-cli
```
